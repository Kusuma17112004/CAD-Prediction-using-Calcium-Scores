Django User & Prediction Management Web App

This is a Django-based web application that manages user details, login history, and prediction inputs. The system allows user registration, image upload, OTP fields, and stores multiple fields used for prediction tasks.

Features

User registration & profile management

Login time tracking

Store prediction input fields

Ready for integrating ML models

Database handled using Django ORM

Tech Stack

Django (Backend)

Python

SQLite/MySQL (via Django ORM)

Optional ML model integration (.pkl file)
