# Machine Learning based Coronary Heart Disease Prediction Management Web App

This is a Machine Learning-based web application that manages user details, login history, and prediction inputs. The system allows user registration, image upload, OTP fields, and stores multiple fields used for prediction tasks.

## Features 
- Store prediction input fields  
- Ready for integrating ML models  
- Database handled using Django ORM
- Maintains login history for each user

## Tech Stack
- Django (Backend)  
- Python  
- MySQL  

## How to Run
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
